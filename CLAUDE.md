# Fireworks Training Cookbook

Ready-to-run training recipes built on the [Fireworks Training SDK](https://github.com/stainless-sdks/fireworks-ai-python) (`fireworks.training.sdk`). Only `training/` is relevant -- ignore other top-level directories.

**Skill:** This repo ships a Claude Code skill in `skills/training/`. Use it to pick recipes, run examples, customize losses and rewards, promote checkpoints to deployable models, re-attach deployments to new trainers, and understand cookbook internals. See `skills/training/references/` for deep dives on specific topics.
